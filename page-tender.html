АНАЛИТИКА и отчетность

Seldon 1.7 содержит базу данных по контрактам и протоколам с 2005 года.

Изучите тактику потенциального конкурента с помощью отчета «Процент снижения цены».

Воспользуйтесь отчетом «Динамика закупок» и оцените спрос и предложения на ваш товар в интересующем регионе.

Воспользуйтесь отчетами «Крупнейшие поставщики» и «Крупнейшие заказчики». Вы получите список главных игроков в интересующий отрасли со всеми контактными данными.

Используйте клиентскую базу для увеличения своих продаж и заключения выгодных сделок.

ПОиск и фильтры

Поиск информации в Seldon осуществляется с помощью встроенных поисковых фильтров. Они анализируют закупки, контракты, площадки и организации, а также отсеивают несоответствующую информацию.

Чтобы задействовать основной фильтр, достаточно выбрать из предложенного списка нужные параметры и указать их значения. Поиск по условиям позволяет находить информацию, в точности соответствующую вашим требованиям.

ПРЕИМУЩЕСТВА Использования системы

Seldon 1.7 содержит огромную базу закупок разного статуса: планируемые, текущие, завершенные, отмененные. В системе охвачены все формы проведения торгов: запрос котировок, предложений, аукционы, конкурсы и т.д.

Данные по закупкам, которые собирает система из различных источников, изначально разнородны. В Seldon вся информация приведена к единому виду для быстроты поиска и обработки.

Все закупки разделены на 5 отдельных списков, с каждым из которых можно работать отдельно. Система содержит списки государственных закупок, закупок по 223-ФЗ, коммерческих, международных и планируемых, реализации имущества

Список закупок представляет собой таблицу, поля которой содержат основные сведения по каждой закупке. Таблицу можно сортировать по любому из полей (наименованию закупки, организатору, торговой площадке, статусам, датам и т.д.).

Для медицинской сферы предусмотрены дополнительные параметры подбора торгов (МНН, ЖНВЛП)
